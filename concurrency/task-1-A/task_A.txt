Рассмотрим пример последовательного исполнения инструкций двумя потоками:
		Поток 1			|		Поток 2
					|	       victim = 1
	      victim = 0		|
	     want[0] = true		|
//здесь произойдет заход в критическую	|
// секцию, поскольку want[1] == false	|
					|	     want[1] = true
					|//здесь произойдет заход в критическую
					|//секцию, поскольку victim == 0

Таким образом, оба потока пройдут в критическую секцию, что приводит к нарушению взаимного исключения
